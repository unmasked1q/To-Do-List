📝 To-Do List App

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript. The application allows users to add, complete, and delete tasks while storing data in the browser's local storage so tasks remain available even after refreshing or reopening the page.


🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Automatic task saving using Local Storage
- Responsive design for desktop and mobile devices
- Clean and modern user interface

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
